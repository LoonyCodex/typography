# typography

## Getting Started

Install dependencies,

```bash
$ npm i
```

Compile rescript,

```bash
$ npm run rescript
```

Build documentation,

```bash
$ npm run dumi
```

Publish documentation via Github Pages,

```bash
$ npm run dumi:publish
```
